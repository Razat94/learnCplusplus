Introduction to C++ Programming

To write a computer program, we must first know what is a program. A computer program is a series of steps that is executed by a computer. A computer program can be:

  • An operating system.
  • A web browser like Mozilla Firefox and Apple Safari
  • An office suite used to write documents or spreadsheets.
  • A video game
  • ...what you will be coding!
    • Source*

    As you may guess, a computer program is written by a programmer. In other words, the programmer (you!) would be typing (i.e. coding) out the steps (the program) for the computer to execute. To write a program, a programmer needs to write the source code in the programming language of their choosing (in our case, C++).

    To learn programming, you will first need:

  • a computer
  • a compiler

    • Every programmer needs a computer to type the code on, and a compiler to compile the C++ code. The latter half might lead you to the next question: What is a compiler??

    To understand what a compiler does, we must first understand how do computers operate. Computers are fairly stupid; they can only understand machine code (i.e. machine language), which is a set of instructions consisting of ones and zeros.

    A single instruction to a computer could look like this:

    00000 10011110

    Source*

    A computer program written in the ones and zeroes would be difficult to read and understand by us humans. Instead, computer programmers choose to keep their sanity by writing source code in a programming language, such as C++, and run a program called a compiler to try and translate the language into something the computer can understand.

    In short, when you write a program in C++, you would execute (or run) your code by running another program (the compiler) that translates your C++ source code into machine code.

    Everything covered in this section may seem confusing as first. But as time goes on, you will begin to understand it all better.

    There are many different ways to compile a C++ program. The simplest way to compile it is by using an Integrated Development Environment (IDE). An IDE contains several tools needed for development, such as a text editor and tools needed to compile programs directly.

    I personally really love Dev-C++. Another official IDE is Visual Studio.

    Once you are able to run the IDE, you are ready to start programming! :)
