Functions in C++ Part 2

As stated in the previous chapter, we pass arguments to functions. The arguments are passed in a way that matched the functions parameters are defined.

Having said that, arguments can be passed by value or passed by reference.

Pass By Value

Arguments that are passed by value pass a copy of the argument to the parameters of a function at the moment the function is called. In short, only copies of the argument are passed to the functions parameters. Modifying the variable within the scope of the function will not effect the values of the variables outside it.
