Data Types

In the last chapter, we wrote our first computer program that outputted the text "Hello, World!" upon execution.

\0 When we outputted the sentence "Hello, World", we outputted a string. A string is a type of data that refers to a string of characters. The string data type is just one of many data type. Different data types can do different things. For now, we'll just focus on outputting different types of data to the console window. In this chapter, we will go over what is in my opinion the five most essential data types in C++, which are:

  • int
  • double
  • char
  • string
  • bool

Let's now go over each one in detail.
