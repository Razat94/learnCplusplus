Data Types

In the last chapter, we wrote our first computer program that outputted the text "Hello, World!" upon execution.

As you may know, computer allocates memory to programs in bits. Programs can use different types of bits to represent data. Different type sizes can have different bit sizes; the more bits a type has, the more distinct values it can represent, but at the same time, also consumes more space in memory.

When we outputted the sentence "Hello, World", we outputted a string. A string is a type of data that refers to a string of characters. The string data type is just one of many data type. Different data types can do different things. For now, we'll just focus on outputting different types of data to the console window. In this chapter, we will go over what is in my opinion the five most essential data types in C++, which are:

  • int
  • double
  • char
  • string
  • bool

Let's now go over each data type in detail.

  • int

    As you may guess, int simply stands for 'integer'. In normal mathematics, integers are known as whole numbers i.e. numbers without a fractional component. In C++, this concept is a bit different. In programming, the int data type is represented by 32 bit memory.
    You can visualize this number by thinking of 32 individual bits. Each bit can be "on" (1) or "off" (0), and because there are 32 bits in an int value, the total number would be 232 = 4,294,967,296 (~4 billion). It is important to keep in mind that we must include negative numbers and the number 0. Thus, the values for the int data type can only go up to +2147483647 & -2147483648.

  • double

    This type represents real values, such as integers AND decimal numbers.

  • char

    This type represents a single character, such as 'Y', 'N' or '@'. This character represents a byte of information.

  • string

    As you may guess, the string data type represents a string of characters. We can use strings like so: cout << "Hello " + "World"

  • bool

    Boolean types can simply represent true or false; 1 or 0. We use the true and false keywords without quotation marks to represent boolean values! A keyword is a reserved word in C++. We will go over it again in the next chapter.

Although there are different data types, we must be careful when dealing and comparing data types. You can add two strings together:
cout << "Hello " + "World!";

& you can add two integers together:
cout << 3 + 4;

But if I try adding two different data tyles like so: cout << ( 3 + "Hello World" ); I should of course get an error. Does it make any sense to add "Hello World!" to the integer three?

Although the last example was a bit intuitive, it can help answer the next question: Does it make any sense to add 3 & "4" together? Like so: cout << 3 + "4"; The answer to that is no as well.
