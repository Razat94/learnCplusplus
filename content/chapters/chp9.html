Functions in C++

A function is a block of code which runs when it is used (i.e. called). You can think of it like a mini-program; each function has a start and an end.

So far, you've been writing code in the main function. As you can guess, there can only be one main function. The main function will always be called at the beginning of execution, but we can create our own functions and place them wherever we'd like inside the main function. In doing so, The main function will call the right functions when executed and pass information to and from the functions it calls.

Syntax

return_type function_name {
  // code to be executed.
}

You may be asking yourself: Well, why do we need functions? Why not just write all our code in the main function? From what I can tell, there are 3 main reasons why we use them:

  • Functions can cut large programs into smaller, manageable pieces.
  • Functions are really handy since once they're written, it can be used and re-used indefinitely.
  • Functions make the program easier to read.
