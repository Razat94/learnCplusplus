Conditional Statements

Before we going over conditional statements, let's first remember what a block of code is.

A code block (or block of code) is literally just some code written inside of { } brackets.

For each block, remember that each and every curly brackets must be in a pair so that they are closed. So:

  • Every opening brace '{' must have a closing brace '}'
  • Every closing brace '}' must have an opening brace '{'.

Now that we know what a block of code is, we can now go over conditional statements.

Syntax:

if (condition) {
  // block of code to be executed if the condition is true
}
