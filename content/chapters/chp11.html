Arrays

Arrays are used to store multiple values in a single variable, instead of declaring separate variables for each value.

So suppose you need to store quiz scores for 10 students in a large lecture class. You could create 10 variables, and have each variable hold each quiz score:

quiz1, quiz2, quiz3, â€¦ quiz10

Additionally, we would read in the scores as so:
cin >> quiz1;
cin >> quiz2;
cin >> quiz3;
...

The problem with this method is that you would need to create a lot of variables that are seemingly repetitive. This is where we can use arrays. To declare an array, we define the array type and specify the specific name (identifier) of the array followed by an integer in square brackets to specify the number of elements it should store: So in our quiz example, we would declare it as so:

int quiz[9]

In the example above:

  • We specifiy the data type of the array as an "int"
  • We give the array just one name "quiz"
  • We state that the array has 10 elements; i.e. the size of the array is 10.

Note: Array indexes start with 0: [0] is the first element. [1] is the second element, etc.

To reiterate what's said above, indexes always begins with index 0! So if I have an array:
    int score [5];
then the array holds 5 integer variables starting with score[0].

Note: There is no score[5]!

BE CAREFUL NOT TO GO PAST THE ARRAYS SIZE!

Each variable is identified and can be accessed by the array name + index (integer). So for example:
quiz[1], quiz[3],

All the components in the array MUST be the same type! So
    double quiz [125];
holds 125 doubles (real numbers).

Overall, this is a good programming rule because the array should hold quiz values and not anything else.
